# Deployment

1. Extract the v1.0 package.
2. In GitHub, delete old application and patch files from the repository root.
3. Upload all files from this package.
4. Commit with message: `Release Iron Disciple OS v1.0.0`.
5. Confirm GitHub Pages uses `main` and `/ (root)`.
6. Wait for the Pages deployment.
7. Open the site in Safari and refresh.
8. Re-add to the iPhone Home Screen if the old cache remains.
