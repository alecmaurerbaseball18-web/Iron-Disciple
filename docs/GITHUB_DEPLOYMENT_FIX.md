# GitHub Pages Deployment Fix

This repository includes an explicit GitHub Pages workflow.

## One-time GitHub setting

1. Open the repository.
2. Go to **Settings → Pages**.
3. Under **Build and deployment**, set **Source** to **GitHub Actions**.
4. Open **Actions** and select **Deploy Iron Disciple to GitHub Pages**.
5. Press **Run workflow** if a deployment did not start automatically.

The site is a static application. No Jekyll, npm, package installation, or build command is required.
