# Phase 1 — Onboarding and Reliability

## Added

- First-run onboarding for profile and training targets.
- Goal sanity warnings for weight and body-fat targets.
- Persistent onboarding completion state.
- Visible setup reminder on the Today dashboard.
- Manual foundation health check.
- Last health-check timestamp and result.
- Updated service-worker cache for the new foundation script.

## Acceptance criteria

- A new user can complete the essential profile from one guided form.
- Data remains available after refresh or relaunch.
- Tournament records are visible and editable.
- Foundation check reports navigation, dashboard, profile, events, storage, backup, offline support, and settings readiness.
- Existing application modules remain intact.
